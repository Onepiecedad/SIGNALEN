Dashboard — Game Master Control

Webbaserad realtidspanel i personalrummet. React + WebSocket till alla RP2040-controllers. Game master ser allt — och kan styra allt.
Realtidsöversikt
Status, tid kvar, lösta puzzles, svårighetsnivå och atmosfärnivå — allt live.
Puzzle-detalj
Namn, status, tid, försök, konsekvenser. Sensordata i realtid. Automatisk varning vid >3 min utan framsteg.
Hint-system
Tre nivåer med AI-förslag. Leverans via stationens AI-röst eller OLED-display direkt i rummet.
Sessionhistorik
Varje session sparas. Trender över tid. Komponenthälsa per enhet för prediktivt underhåll.
